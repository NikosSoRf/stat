# stat
Код для реализации классификаторов аритмий

Перед запуском проекта распакуйте папку data и установите все необходимые библиотеки и инструменты (полный список зависимостей представлен в файле requirements.txt).

Для получения результатов работы алгоритмов SVM, KNN, Naive Bayes запустите код main.py, результирующие метрики будут сведены в файл Exsel model_comparison ( файл создается в папке проекта и в последующих запусках перезаписывается)

Запуск main2.py со строкой data = comb.comb() выведет результат работы коллективов алгоритмов с нормализованной малой выборкой, если переключить строку на data = comb.augment_data() алгоритмы будут обучаться на аугментированной выборке ( по 212 записей на класс). Результаты работы перезаписывают файл model_comparison. 

Запуск main2Test.py по функционалу схож с main2.py, но дополнительно выводит статистические тесты ( кросс-валидацию 10-fold CV, доверительные интервалы, t-тесты) в терминал.

Важно заметить, что результаты временных параметров сильно зависят от вычислительной способности устройства.
